# Support

For ZeticMLange Flutter SDK help:

- Read the documentation: https://docs.zetic.ai
- Check troubleshooting: https://docs.zetic.ai/troubleshooting/common-errors
- Open an issue in this repository for package bugs, documentation gaps, and
  platform integration problems.

For account, dashboard, or commercial support, contact ZETIC through the support
channels listed in the documentation.
