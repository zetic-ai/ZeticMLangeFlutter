# ZeticMLange Flutter

Public support repository for the ZeticMLange Flutter SDK.

The Flutter package is published as `zetic_mlange` and provides Dart APIs for
running ZeticMLange on-device AI models on Android and iOS.

## Package

```yaml
dependencies:
  zetic_mlange: ^1.0.0
```

```dart
import 'package:zetic_mlange/zetic_mlange.dart';
```

## Issues

Use this repository to report issues with the published Flutter SDK package,
documentation, examples, or platform integration.

When reporting a runtime issue, include:

- SDK package version
- Flutter and Dart versions
- Target platform and OS version
- Device model
- Relevant error logs
- Minimal code or steps to reproduce

## Source Code

The SDK source is maintained in a private repository. This public repository is
used for package metadata, issue tracking, release notes, and support.
